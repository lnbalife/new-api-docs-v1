<Callout type="info" title="Chat Settings Option">
  In the New API console under **System Settings → Chat Settings**, you can add the following shortcut so that the token management page can fill CC Switch with one click:
</Callout>

```json
{
  "CC Switch": "ccswitch"
}
```

## Fill-in Steps

1. **One-click fill**  
   On the token management page, find the template above and click **CC Switch** to fill in with one click.

2. **Click New API**  
   After filling, click **New API**. A **「填入 CC Switch」** (Fill in CC Switch) configuration dialog will open.

3. **Configure in the dialog**
   - **应用 (Application)**: Choose the app type (Claude / Codex / Gemini). Claude is selected by default.
   - **名称 (Name)**: Enter an application name (e.g. «My Claude»).
   - **Models** (all dropdowns):
     - **主模型 \*** (Main Model — required)
     - **Haiku 模型** (Haiku Model)
     - **Sonnet 模型** (Sonnet Model)
     - **Opus 模型** (Opus Model)  
   When nothing is selected, they show «请选择模型» (Please select model). Choose the corresponding model in each dropdown as needed.

4. **Finish**  
   After configuration, click **「打开 CC Switch」** (Open CC Switch) to use it, or **「取消」** (Cancel) to close without saving.
