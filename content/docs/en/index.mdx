## Overview

New API provides a complete set of RESTful AI Model APIs. These interfaces allow you to invoke various AI capabilities, compatible with the OpenAI API format.

<Callout title="Online Debugging">
  You can visit [Apifox Playground](https://apifox.newapi.ai/) to test and debug API interfaces online, or browse the API documentation below.
</Callout>
