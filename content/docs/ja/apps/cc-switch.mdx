<Callout type="info" title="チャット設定オプション">
  New API コンソールの **システム設定 → チャット設定** で、以下のショートカットを追加すると、トークン管理ページから CC Switch にワンクリックで入力できます：
</Callout>

```json
{
  "CC Switch": "ccswitch"
}
```

## 入力手順

1. **ワンクリック入力**  
   トークン管理ページで上記テンプレートの **CC Switch** をクリックしてワンクリック入力します。

2. **New API をクリック**  
   入力後、**New API** をクリックすると「填入 CC Switch」設定ダイアログが開きます。

3. **ダイアログで設定**
   - **应用 (アプリ)**：アプリタイプ（Claude / Codex / Gemini）を選択。既定は Claude。
   - **名称 (名前)**：アプリ名を入力（例：「My Claude」）。
   - **モデル**（いずれもドロップダウン）：
     - **主模型 \***（メインモデル・必須）
     - **Haiku 模型**
     - **Sonnet 模型**
     - **Opus 模型**  
   未選択時は「请选择模型」と表示されます。各ドロップダウンで必要なモデルを選択してください。

4. **完了**  
   設定後 **「打开 CC Switch」** をクリックして利用開始。**「取消」** でキャンセルできます。
