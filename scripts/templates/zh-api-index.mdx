## 概述

NodeKey 提供完整的 RESTful API 接口，分为 **AI 模型接口** 和 **管理接口** 两大类。您可以通过这些接口实现 AI 能力调用和系统管理功能。

## AI 模型接口

AI 模型接口提供各种 AI 能力的调用，兼容 OpenAI API 格式。
